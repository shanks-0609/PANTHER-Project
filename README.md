# PANTHER-Project
PANTHER: AI-Driven Pancreatic Tumor Segmentation on MRI 3D Attention U-Net for pancreatic tumor segmentation on MRI (PANTHER challenge). Includes self-supervised pretraining (Models Genesis), differential-LR transfer learning from diagnostic T1 MRI to MR-Linac T2 MRI, combined Dice+CE loss, and MONAI-based pipeline. Built with PyTorch &amp; MONAI.
